# 🔐 Advanced Password Generator

A modern and responsive Password Generator Web Application built using **HTML, CSS, and JavaScript**. This application allows users to generate secure passwords with customizable options such as uppercase letters, lowercase letters, numbers, special characters, and password length.

## 📖 Project Description

This project was developed to demonstrate core web development skills, including:

* HTML page structure
* CSS styling and responsive design
* JavaScript DOM manipulation
* Event handling
* Dynamic password generation
* Password strength analysis
* Clipboard functionality

The application generates random passwords based on user-selected criteria and provides real-time feedback on password strength.

---

## ✨ Features

* Generate random secure passwords
* Adjustable password length
* Include uppercase letters (A-Z)
* Include lowercase letters (a-z)
* Include numbers (0-9)
* Include special characters (!@#$%^&*)
* Password strength meter
* Copy password to clipboard
* Responsive design for desktop and mobile devices
* Modern Glassmorphism-inspired user interface
